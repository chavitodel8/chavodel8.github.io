Ejercicio 10: Suma de arrays

Problema: Dado dos arrays de igual longitud, devuelve un array con la suma de sus elementos en cada posición.


function sumarArrays(a, b) {
    return a.map((val, i) => val + b[i]);
}
console.log(sumarArrays([1, 2, 3], [4, 5, 6])); // [5, 7, 9]

Ejercicio 2: Invertir cadena

Problema: Escribe una función que invierta una cadena de texto.


function invertirCadena(str) {
    return str.split('').reverse().join('');
}
console.log(invertirCadena("Hola")); // "aloH"
